Последнее значение цикла


let i = 3;

while (i) {
  alert( i-- );
} // Выведенным значением будет 1, так как в условии цикла while по умолчанию i!=0



Какие значения выведет цикл while?


Постфиксный вариант:
let i = 0;
while (i++ < 5) alert( i ); // от 1 до 5, так как сначала сравнивается присвоенное число и только потом оно увеличивается

Префиксный вариант:
let i = 0;
while (++i < 5) alert( i ); // от 1 до 4, так как сначала число увеличивается и только потом сравнивается



Какие значения выведет цикл for?


Постфиксная форма:
for (let i = 0; i < 5; i++) alert( i ); // от 0 до 4

Префиксная форма:
for (let i = 0; i < 5; ++i) alert( i ); // от 0 до 4



Замените for на while


Было:
for (let i = 0; i < 3; i++) {
  alert( `number ${i}!` );
}

Стало:
let i = 0
while (i < 3) {
    alert( `number ${i}!`;
    i++;
}
